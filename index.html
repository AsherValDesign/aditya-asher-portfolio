CPM Schedule — Primavera P6

Developed using Critical Path Method to model project sequencing and duration. Activities were structured with WBS logic ties reflecting realistic construction dependencies, identifying the critical path and long-lead procurement items.

Financial Modeling NPV / IRR Earned Value Management Sitework Estimating

This project developed a comprehensive construction and financial plan for a commercial mall, focusing on delivery strategy, cost forecasting, and long-term financial performance. It began with conceptual estimating using city cost indices and inflation adjustments to establish an accurate baseline budget.

Project Delivery Method Analysis

Analyzed and compared Design-Bid-Build, Design-Build, and CM-at-Risk delivery methods to evaluate risk allocation, cost control, and schedule efficiency. Developed recommendations based on project complexity and financial exposure.

Project Financial Model

Evaluated long-term feasibility through detailed cash flow forecasting incorporating equity investment, loan financing, operating costs, debt service, and projected rental revenue. Calculated NPV and IRR using discounted cash flow methods to assess investor return potential.

Sitework Estimate & Production Analysis

Production-based cost model integrating labor crews, equipment hourly costs, material quantities, and activity durations. Productivity rates (LCY/hr) and unit rate development were used to simulate contractor pricing strategy.

Earned Value Management
